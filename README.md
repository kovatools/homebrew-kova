# Homebrew Tap for Kova CLI

Official [Homebrew](https://brew.sh) tap for the [Kova](https://kovatools.com) CLI.

## Install

```bash
brew install kovatools/kova/kova
```

## Upgrade

```bash
brew upgrade kova
```
